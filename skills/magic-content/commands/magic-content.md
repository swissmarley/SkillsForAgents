Invoke the `magic-content` skill via the Skill tool. The skill drives the full guided menu: topic input, source discovery (YouTube/Web/PDF), source selection, artifact selection, NotebookLM run, artifact review, and wiki ingest.

If the user passed arguments after `/magic-content`, treat them as the topic and skip the topic-input prompt. Otherwise, the skill will ask for the topic.
